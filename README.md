# Python-REST-API-with-Flask-Inventory-Management-System

This is an app built using Flask, HTML and JavaScript for the summative lab: <br>
https://moringa.instructure.com/courses/1389/assignments/87559/

## Installation & Setup
- Use `pipenv install`  on the terminal to install the required dependencies.
- Run `pipenv shell` to activate the virtual environment.

## Running the app
- In the terminal, run `python app.py` 
- Open the `index.html` file in a browser to view and interact with the frontend.
